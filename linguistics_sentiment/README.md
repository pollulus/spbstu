## Программа для выполнения анализа тональности отзывов и построения графика результатов
- **Датасет** `cafe_reviews.csv` содержит отзывы о кафе. 
- Посредством предобученной модели <ins>blanchefort/rubert-base-cased-sentiment</ins> выполнен **анализ тональности** отзывов из датасета.
- С помощью библиотеки <ins>Matplotlib</ins> создана **столбчатая диаграмма**, отражающая количество отзывов для каждой категории тональности.
