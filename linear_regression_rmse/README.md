## Программа для обучения модели линейной регрессии
- **Датафрейм** `data.csv` содержит данные о характеристиках автомобилей и их стоимости
- Построена **модель линейной регрессии** для предсказания цен автомобилей
- Точность модели **RMSE** сравнивается с показателем **"наивной" модели**
- Обученная модель ошибается в среднем на **40 000** при предсказании цены автомобиля
- Наивная модель ошибается в среднем на **56 000**
- Разница составляет около **16 000**, именно на столько обученная модель точнее предсказания среднего значения цены
