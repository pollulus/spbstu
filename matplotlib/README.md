## Программа для визуализации данных
- Датафреймы `df_2023` и `df_2024` созданы с помощью библиотеки <ins>Pandas</ins> на основе данных о прочитанных книгах из файла *books.csv*
- **Графики** двух датафреймов созданы и размещены на одном холсте с помощью библиотеки <ins>Matplotlib</ins>
- Программа представляет пример настройки параметров линейного графика и столбчатой диаграммы
