## Программа для поиска самой длинной подстроки без повторений в списке любых элементов
Программа состоит из трех **функций**:
- `check_input` проверяет элемент массива и возвращает True, если элемент – строка, иначе – False
- `get_substring` находит и возвращает самую длинную подстроку без повторений в переданной строке
- `main` принимает на вход список, находит его самую длинную строку, передает ее на метод get_substring и печатает индекс самой длинной строки в массиве и найденную подстроку
### Пример работы
- Входные данные: [[0, 4, 8], ‘abca’, 7, ‘aaaaa’]
- Выходные данные: 3, 'a'
