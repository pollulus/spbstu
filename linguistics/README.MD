## Программа для извлечения ключевых слов

CSV-файл с отзывами
ф-я find_keywords() принимает путь к файлу

Загружает данные из указанного CSV-файла в DataFrame.
Для каждого предобработанного отзыва (столбец «normalized_text») извлекает ключевые слова и оценку уверенности модели для каждого из них.
Результаты сохраняются в виде списка кортежей в новый столбец «key_words». Для извлечения используйте один из предложенных в лекции вариантов.
Возвращает обновленный DataFrame.
