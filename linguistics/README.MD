## Программа для извлечения ключевых слов
- **Датафрейм** представляет собой CSV-файл с отзывами о кафе.
- Отзывы предобаботаны: проведена **лемматизация** и **удаление стоп-слов**.
- Для каждого отзыва с помощью `библиотеки YAKE` по частотности извлечены ключевые слова и представена оценка уверенности модели для каждого из них. Данные записаны в столбец `«key_words»`.
